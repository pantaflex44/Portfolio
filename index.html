coding life

Tout à commencer en 1999, année où j'ai découvert le monde informatique. Un monde étrange, virtuel, animé par des personnes toutes aussi extraordinaires que leurs mots.

L'informatique, doux nom donné à un domaine complètement fou, où l'on pouvait créer, manipuler, partager, sans aucune limite autre que l'innovation de son temps.

Du matériel au logiciel, je me suis rapidement pris de passion par tous ces 1 et 0. D'abord sur les illustres Thomson et Amstrad, je me suis vite tourné vers les PCs.

Windows 95, puis 98, 2000 (une référence à mon sens), j'ai essoré mes machines jusqu'à découvrir les Amériques: Linux!

Ok, mais le code dans tout ça?

La passion du code arrivera plus tard malgré mes début sur QBasic. Il m'aura fallu attendre de découvrir Visual Basic Script via Excel pour enfin passer du coté obscure de la barrière.

S'est enchainé, Visual Basic 6, Pascal (lui, ce n'est pas mon pôte), un semblant de Java pour me rendre compte que l'avenir allait se passer dans le monde des Sims: Internet.

Du local à l'internationnal

Les années passent, .Net et son C# d'un coté, mon adoption par les pingouins, Python, la relève, je finis par me tourner vers ce qui allait tous nous réunir.

Et là, c'est le drâme! HTML, CSS, les flocons de neige de Javascript, puis El Famoso PHP finiront par me séquestrer pour toujours.

Parce que Javascript était pour moi un calvaire, que j'ai décidé de m'y coller, pour aujourd'hui choisir par défaut, de développer avec NodeJS et mon préféré: ReactJS!

Et sinon...

numéro hors série

Oui, mais non. Ca restera avant tout une passion.

Mon métier: Electricien

Après avoir plus ou moins récemment essayé de transformer ma passion en métier, en commencant par une formation type Graduate Dev Web / Web mobile, je me suis rendu compte que je ne voulais pas me retrouver enfermé dans une bulle bien réglée.

Le code restera donc un moyen de me faire plaisir, de me défier, d'apprendre, de partager, de rendre service (pour exemples, le site Internet de l'entreprise AZ RESO ou feu, le média libre Born to be jaune)

Petits plus

Mecanique automobile, Cinéma / Séries, Bricolage, quelques passe-temps suplémentaires complétant un quotidien bien vivant :-)

Vous l'aurez compris, le code fait partie intégrante de ma vie. Un équilibre.
